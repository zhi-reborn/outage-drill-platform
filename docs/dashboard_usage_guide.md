## 🎯 问题分析

### 为什么流程步骤可视化部分为空白？

**原因**: 没有演练实例数据

指挥中心大屏需要先有演练实例才能显示流程步骤。系统流程：
```
模板管理 → 创建演练模板
    ↓
演练管理 → 创建演练实例
    ↓
指挥中心 → 选择演练 → 启动演练
    ↓
流程步骤可视化 → 显示步骤执行状态
```

## 🎯 使用流程

### 完整演练流程

```
步骤1: 查看模板
    ↓
进入管理后台 → 模板管理
    ↓
查看"标准灾备演练流程"模板(已存在)
    ↓

步骤2: 创建演练
    ↓
进入管理后台 → 演练管理
    ↓
点击"新增演练"
    ↓
选择模板 → 输入演练名称 → 创建
    ↓

步骤3: 启动演练
    ↓
返回指挥中心大屏
    ↓
选择刚创建的演练
    ↓
点击"启动演练"(在演练管理页面)
    ↓

步骤4: 监控进度
    ↓
流程步骤可视化显示步骤状态
    ↓
实时查看步骤执行情况
    ↓
分配任务给参演人员
    ↓

步骤5: 完成演练
    ↓
参演人员执行任务
    ↓
指挥中心实时监控
    ↓
演练完成 → 结束演练
```

## 🆘 常见问题

### Q: 为什么看不到流程步骤？
**A**: 需要先创建演练实例。点击"创建演练"按钮，在演练管理页面创建演练。

### Q: 创建演练后还是看不到？
**A**: 
1. 返回指挥中心大屏
2. 在下拉菜单中选择演练
3. 如果演练状态是"待启动"，需要先启动演练

### Q: 如何启动演练？
**A**: 
1. 进入管理后台 → 演练管理
2. 找到创建的演练
3. 点击"启动"按钮
4. 返回指挥中心查看进度

### Q: 控制台显示什么？
**A**: 
打开F12查看控制台，会显示详细的数据加载过程和当前状态。
