# 管理后台导航功能说明

## ✅ 新增功能

已为管理后台添加了完整的导航功能，方便快速切换到其他页面。

---

## 🎯 导航方式

### 方式1: 顶部导航菜单
**位置**: 页面顶部Header区域

**菜单项**:
- 📊 **指挥中心大屏** - 快速跳转到指挥中心
- 🔧 **参演工作台** - 快速跳转到工作台
- 🐛 **系统诊断** - 快速跳转到诊断页面
- ⚙️ **管理后台** - 当前页面（高亮显示）

**特点**:
- ✅ 深色主题，与Header融为一体
- ✅ 当前页面自动高亮
- ✅ 一键切换，无需返回主页

---

### 方式2: 左侧快捷导航
**位置**: 左侧Sider菜单顶部

**菜单分组**:
```
快捷导航
├─ 📊 指挥中心大屏
└─ 🔧 参演工作台

─────────────

管理功能
├─ 👥 用户管理
├─ 📄 模板管理
├─ ⚡ 演练管理
└─ 🔗 Webhook配置
```

**特点**:
- ✅ 分组清晰，易于识别
- ✅ 快捷导航在最上方
- ✅ 管理功能在下方
- ✅ 分隔线区分不同功能区域

---

### 方式3: 页面标题栏快捷链接
**位置**: 内容区域顶部标题栏右侧

**快捷链接**:
- 📊 **指挥中心** - 点击跳转
- 🔧 **参演工作台** - 点击跳转

**特点**:
- ✅ 在每个管理页面都显示
- ✅ 右侧位置，方便操作
- ✅ 带图标，直观明了

---

### 方式4: 用户下拉菜单

**菜单项**:

## 🎯 使用场景

### 场景1: 从管理后台快速查看演练进度
**操作流程**:
```
管理后台 → 模板管理
    ↓
点击顶部"指挥中心大屏"
    ↓

    ↓

```

**操作流程**:
```
管理后台 → 演练管理 → 创建演练
    ↓
点击左侧"指挥中心大屏"
    ↓
选择演练 → 启动演练
```

### 场景3: 管理员执行任务
**操作流程**:
```
管理后台 → 用户管理
    ↓
点击页面标题栏"参演工作台"
    ↓
查看并执行分配的任务
```

## 🔧 技术实现

### 顶部导航菜单
```typescript
<Menu
  mode="horizontal"
  selectedKeys={[当前路径]}
  items={topMenuItems}
  onClick={handleMenuClick}
  theme="dark"
/>
```

### 左侧分组菜单
```typescript
<Menu
  mode="inline"
  items={[
    { type: 'group', label: '快捷导航' },
    { key: '/dashboard', label: '指挥中心大屏' },
    { type: 'divider' },
    { type: 'group', label: '管理功能' },
    { key: '/admin/users', label: '用户管理' },
  ]}
/>
```

### 页面标题栏
```typescript
<div style={{ display: 'flex', justifyContent: 'space-between' }}>
  <div>页面标题 + 描述</div>
  <Space>
    <a onClick={() => navigate('/dashboard')}>指挥中心</a>
    <a onClick={() => navigate('/workbench')}>参演工作台</a>
  </Space>
</div>
```

## 🆘 常见问题

### Q: 看不到顶部导航？
**A**: 强制刷新浏览器(Ctrl+Shift+R)

### Q: 点击菜单没有反应？
**A**: 检查浏览器控制台是否有错误

### Q: 左侧菜单没有分组？
**A**: 确认已刷新页面，检查代码是否更新

### Q: 页面标题不显示？
**A**: 检查路由配置，确认子页面正确加载
